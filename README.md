# App de Clima

Aplicação web em TypeScript com Vite para visualizar informações meteorológicas de forma simples e intuitiva.

## 🚀 Tecnologias

- Vite
- TypeScript
- CSS puro

## 📋 Pré-requisitos

Antes de começar, certifique-se de ter instalado:

- Node.js 18 ou superior
- npm

## ▶️ Como executar

No diretório do projeto, execute:

```bash
npm install
npm run dev
```

A aplicação ficará disponível no navegador em uma URL local fornecida pelo Vite.

## 🛠️ Scripts disponíveis

- `npm run dev` — inicia o servidor de desenvolvimento
- `npm run build` — gera a versão de produção
- `npm run preview` — preview da build local

## 📁 Estrutura do projeto

```text
src/
  main.ts
  style.css
  services/
  types/
  ui/
  utils/
```
